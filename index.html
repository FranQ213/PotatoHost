PotatoHost — uruchom serwer Minecraft w Codespaces

Kliknij poniżej, aby otworzyć stronę tworzenia Codespace z tym repozytorium. Codespace będzie zawierał gotowy skrypt start.sh, który obsługuje Java / Paper i odpali serwer.

🚀 Stwórz Serwer 📂 Repozytorium na GitHub
Uwaga: Na jednym koncie GitHub może być nieograniczona liczba serwerów, ale tylko 2 aktywne! Przestrzenie kodów są uruchamiane na koncie użytkownika i mogą mieć timeout/rozliczenia - nie jest to serwer hostingowy działający 24/7 bez dodatkowych działań.

Jak to działa

  1. Repo zawiera .devcontainer i start.sh.
  2. Kliknięcie „Stwórz Serwer” otwiera GitHub — tworzysz Codespace na swoim koncie.
  3. Wpisujesz dwie komendy które będą podane, wybierasz silnik i wersje serwera, a nastęnie instaluje się JDK.
  4. Aby podłączyć się do serwera z Internetu, użyj MineKube Connect / własnego tunelu / VPS.
