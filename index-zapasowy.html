PotatoHost — uruchom serwer Minecraft w Codespaces

Kliknij poniżej, aby otworzyć stronę tworzenia Codespace z tym repozytorium. Codespace będzie zawierał gotowy skrypt start.sh, który obsługuje Java / Paper i odpali serwer.

🚀 Stwórz Serwer 📂 Repozytorium na GitHub
Uwaga: drugie Codespace wymaga konta GitHub. Przestrzenie kodów są uruchamiane na koncie użytkownika i mogą mieć timeout/rozliczenia - na serwerze nie będącym serwerem hostingowym 24 godziny na dobę, 7 dni w tygodniu.

Jak to działa

  1. Repo zawiera .devcontainer i start.sh.
  2. Kliknięcie „Stwórz Codespace” otwiera GitHub — tworzysz Codespace na swoim koncie.
  3. Devcontainer automatycznie uruchamia skrypt, który pobiera Paper i instaluje JDK.
  4. Aby podłączyć się do serwera z Internetu, MineKube Connect / własnego tunelu / VPS.

Szybkie komendy (dla twórcy)

# uruchom lokalnie start.sh (test) WERSJA_KOMÓRKI=1.21.8 PAMIĘĆ_KOMÓRKI=2 ./start.sh # utwórz Codespace ręcznie https://codespaces.new/FranQ213/PotatoHost

Chcesz automatyzację (ustawianie tokenów, jedno kliknięcie)? To wymaga backendu i OAuth — mogę przesłać później.
