PotatoHost — uruchom serwer Minecraft w Codespaces

Kliknij poniżej, aby otworzyć stronę tworzenia Codespace z tym repozytorium. Codespace będzie zawierać gotowy skrypt start.sh, który przygotuje Java / Paper i odpali serwer.

🚀 Stwórz Codespace 📂 Repozytorium na GitHub
Uwaga: utworzenie Codespace wymaga konta GitHub. Codespaces są uruchamiane w koncie użytkownika i mogą mieć timeout/rozliczenia - to nie jest serwer hostingowy 24/7.

Jak to działa

  1. Repo zawiera .devcontainer i start.sh.
  2. Kliknięcie „Stwórz Codespace” otwiera GitHub — tworzysz Codespace w swoim koncie.
  3. Devcontainer automatycznie uruchamia skrypt, który pobiera Paper i instaluje JDK.
  4. Aby podłączyć się do serwera z Internetu, użyj MineKube Connect / własnego tunelu / VPS.

Szybkie komendy (dla twórcy)

# uruchom lokalnie start.sh (test)
MINE_VERSION=1.21.8 MINE_RAM=2 ./start.sh

# utwórz Codespace ręcznie
https://codespaces.new/FranQ213/PotatoHost

Chcesz automatyzację (ustawianie tokenów, jedne klik)? To wymaga backendu i OAuth — mogę to dodać później.
