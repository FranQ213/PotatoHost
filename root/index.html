PotatoHost — uruchom serwer Minecraft w Codespaces

Kliknij poniżej, aby otworzyć stronę tworzenia Codespace z tym repozytorium. Codespace będzie zawierać gotowy skrypt start.sh, który przygotuje Java / Paper i odpali serwer.

🚀 Stwórz Serwer 📂 Repozytorium na GitHub
Uwaga: otwarcie kreatora z automatyzacją (OAuth/API) wymaga backendu; bez backendu przycisk otworzy lokalny/plikiowy kreator (jeśli istnieje) — w przeciwnym razie zostaniesz przekierowany do GitHub Codespaces jako fallback.

Jak to działa

  1. Repo zawiera .devcontainer i start.sh.
  2. Kliknięcie „Stwórz Serwer” otwiera nową kartę z lokalnym kreatorem — to Twoje UI (nie GitHub UI).
  3. Kreator może przeprowadzić OAuth z GitHub i użyć API do utworzenia Codespace / repo dla użytkownika.
  4. Devcontainer automatycznie uruchamia skrypt, który pobiera Paper i instaluje JDK.
  5. Aby podłączyć się do serwera z Internetu, użyj MineKube Connect / własnego tunelu / VPS.

Szybkie komendy (dla twórcy)

# uruchom lokalnie start.sh (test)
MINE_VERSION=1.21.8 MINE_RAM=2 ./start.sh

# utwórz Codespace ręcznie
https://codespaces.new/FranQ213/PotatoHost

Jeśli chcesz, mogę dodać automatyzację (OAuth + backend) żeby tworzenie Codespace/repo działało w jednym kliknięciu bez przechodzenia do GitHub — proste rozwiązanie jest dostępne w repozytorium (create.html + server.js).
